# 🍎 Bhuramal Grocery — Premium Online Store

![Bhuramal Grocery Mockup](bhuramal_grocery_mockup_1773635281390.png)

Welcome to the official repository for **Bhuramal Grocery**, a modern, high-performance e-commerce platform built for fresh produce and household essentials.

## ✨ Key Features

- **🛍️ Seamless Shopping**: Browse fresh vegetables, fruits, dairy, and spices with a fast, responsive interface.
- **🔐 Secure Authentication**: Integrated Firebase Auth with Google and Email login.
- **🛠️ Admin Dashboard**: Dedicated management portal to track orders, manage inventory, and monitor revenue.
- **💳 Flexible Payments**: Supports both Cash on Delivery (COD) and secure UPI QR code payments.
- **⚡ Built with Next.js**: Utilizing the latest App Router for lightning-fast performance and SEO.

## 🚀 Technical Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (React)
- **Database**: [Firebase Firestore](https://firebase.google.com/products/firestore)
- **Auth**: [Firebase Authentication](https://firebase.google.com/products/auth)
- **Styling**: Modern CSS Modules with Deep Green & Gold aesthetic.
- **Hosting**: Optimized for [Vercel](https://vercel.com).

## 📂 Project Structure

In this modern Next.js project, the **main entry point (homepage)** is located at:
👉 `src/app/page.tsx`

When deployed to Vercel, this file automatically becomes the `index.html` of your website.

## 🛠️ How to Run Locally

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env.local` with your Firebase credentials.
4. Run the development server:
   ```bash
   npm run dev
   ```
